# GameMerge

GameMerge 是一个汇集了各种有趣网页游戏的平台。在这里，你可以找到并玩到各种经典和创新的游戏。

## 功能特点

- 游戏分类浏览
- 响应式设计，支持各种设备
- 全屏游戏模式
- 游戏标签系统
- 简洁美观的用户界面

## 技术栈

- Vue 3
- Vue Router
- Tailwind CSS
- Vite

## 开发环境设置

1. 克隆项目
```bash
git clone https://github.com/yourusername/gamemerge.git
cd gamemerge
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```

## 项目结构

```
gamemerge/

```
